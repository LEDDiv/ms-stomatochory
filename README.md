# Function of Seed Dispersal by Stomatochory

## Overview

This repository implements a simulation framework (function `Macaw_Kernel`) for modeling **seed dispersal by stomatochory**, focusing on a large frugivorous birds (macaws). 

The output is a set of **seed dispersal distances**, representing dispersal kernels under different ecological scenarios.
